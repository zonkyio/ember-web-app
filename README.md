This Ember addon helps you configure and manage the Web App Manifest to create a Progressive Web App.

## Compatibility

- Ember.js v3.24 or above
- Ember CLI v3.24 or above
- Node.js v12 or above

## Installation

```
ember install ember-web-app
```
